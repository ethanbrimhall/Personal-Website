Kahoot Clone

Summary

The goal of this project was to challenge myself in making a replica of the famous Kahoot.it website. For this I used Node.js and MongoDB so that connections from client to server would be fast. The final version of my project included all the main features from the real Kahoot such as the ability to have multiple live games at the same time and to create your own game that you can host.
