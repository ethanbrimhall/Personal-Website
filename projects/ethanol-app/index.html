Ethanol Quiz iOS App

The goal of this project was to create an app for a high school project. The iOS app is a quiz game where the user reads about ethanol, then they try and answer questions about what they read. They are given a score which is stored in Game Center so they can compete against friends. The app category is education/game. (the app is no longer on the app store due to not renewing my developer membership)
